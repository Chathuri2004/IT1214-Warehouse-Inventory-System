
 IT1214-Warehouse-Inventory-System

 Project Description
This is a simple Warehouse Inventory Management System developed using Java. It allows users to manage inventory items such as adding, removing, updating, searching, and displaying items.

  Features
- Add new item
- Remove item
- Update item quantity
- Search item by ID or name
- Display all items

  Technologies Used
- Java
- Java Collections Framework (ArrayList)

  How to Run
1. Compile the files:
   javac Warehouse.java

2. Run the program:
   java Warehouse
